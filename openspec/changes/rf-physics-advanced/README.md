# rf-physics-advanced

Advanced RF physics for the CPU core: multi-edge diffraction (Bullington + Deygout), UTD
wedge diffraction, Jones-vector polarization (mismatch loss + depolarizing reflection),
per-path Doppler shift, and a multipath (reflections + diffraction) coverage mode. Every
effect is additive and default-neutral, so archived Phase 1–7 results stay bit-for-bit
identical unless a feature is explicitly enabled.
